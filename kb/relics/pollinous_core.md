# Pollinous Core

| Field | Value |
|-------|-------|
| Rarity | Event |

## Description

Pollinous Core is an Event Relic from the Colossal Flower event. Every 4 turns, it will draw 2 additional cards at the start of the turn.

## Notes

### Sources

Can only be obtained from the Colossal Flower event by selecting the  option twice and then , losing a total of 18 HP.
