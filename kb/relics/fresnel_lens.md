## Description

Fresnel Lens is an Event Relic from the Drowning Beacon event. It adds Nimble 2 to future block cards.

## Notes

### Sources

Can only be obtained from the Drowning Beacon event by selecting the  option and losing 13 Max HP.
### Interactions

### Synergies
- Cards that apply block multiple times, such as Evil Eye, Second Wind,  Dodge and Roll, Glitterstream,  and Death's Door, benefits from the Nimble enchantment each time block is applied
