## Description

Crossbow is an Ancient Relic. The description is somewhat misleading, as the Energy-reduction effect will be lost once played. Even if it's the same turn, if the affected card is played and then returned to the hand by any means (e.g. using Hologram), the card will be at its original cost.

## Notes

### Sources

At the start of Act 3, can be offered by Tanx.
