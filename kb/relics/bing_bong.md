## Description

Bing Bong is an Event Relic from the Doll Room event. Whenever you add a card to your Deck, it will add one additional copy.

## Notes

### Sources

Can only be obtained from the Doll Room event.
### Interactions

- Bing Bong has synergy with Pael's Tooth. If Pael's Tooth is active while you have Bing Bong, any removed cards will be returned with two copies of the upgraded card.
### Trivia

- Bing Bong is a reference to the character of the same name and appearance from the game PEAK.
- Bing Bong was an imaginary friend in the movie Inside Out.
