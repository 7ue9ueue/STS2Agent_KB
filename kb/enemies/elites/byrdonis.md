# Byrdonis

## Overview

**Byrdonis** is an Elite found in the Overgrowth.

## Stats

### Byrdonis
- **HP**: 81-84 (Asc8+: 90)
- **Appears in**: Debut Overgrowth

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Swoop |  | Deals 17 damage. |
| _(Asc variant)_ | | Deals 19 (Asc9+) damage. |
| Peck |  | Deals 3x3 damage. |
| _(Asc variant)_ | | Deals 4x3 (Asc9+) damage. |

## Pattern

Byrdonis alternates between its two moves in a fixed pattern:
1. Swoop.
1. Peck.
1. Back to Swoop, and so on.
## Notes

- Peck hits 3 times, so each point of Strength adds 3 extra damage total on Peck turns.
