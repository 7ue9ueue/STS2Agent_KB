# Decimillipede

## Overview

**Decimillipede** is an Elite Enemy found in the Hive. It appears as three identical segments that fight together.

## Stats

### Decimillipede
- **HP**: 40-46 (Asc8+: 46-52)
- **Appears in**: Debut Hive

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Bulk |  | Deals 6 damage. Gains 2 Strength. |
| _(Asc variant)_ | | Deals 7 (Asc9+) damage. Gains 2 Strength. |
| Writhe |  | Deals 5x2 damage. |
| _(Asc variant)_ | | Deals 6x2 (Asc9+) damage. |
| Outgas |  | Deals 8 damage. Applies 1 Weak. |
| _(Asc variant)_ | | Deals 9 (Asc9+) damage. Applies 1 Weak. |
| Reattach |  | Revives with 25 HP (if other segments are alive). |

## Pattern

Each segment cycles through Bulk, Writhe, and Outgas in that order. Each segment starts on a different move, so every turn the three segments are always using three different moves.

When a segment is killed, it uses Reattach to revive with 25 HP if at least one other segment is still alive. After reviving, it resumes with a random move.
## In Party With

### Hive
- 2 other Decimillipede segments.

## Strategy Notes

<!-- Reserved for the meta-agent. Append lessons, combos, and heuristics here.
     Wiki facts live above the divider. -->
- Treat as a top-tier elite threat, possibly the strongest elite for Ironclad; avoid Act 2 elites if the deck lacks AoE, strong multi-target damage, or a solid block/draw engine.
- Avoid staggered kills unless you can finish the last live segment quickly; reattach can restore killed segments to 25 HP while any segment remains alive.
- AoE should ideally lower all segments together, then burst-kill the final group before the first dead segment reattaches.
- If one segment dies early, immediately switch to a reattach-timer plan: either kill all remaining segments before it returns or stop spending burst on single-target kills.
- In a delayed stagger plan, spend Vulnerable/potion damage on the segment that lets one attack kill it, then reserve Bash/Pommel/Dismantle-style follow-up for the final segment before reattach.
- Heart of Iron plus Unmovable/Second Wind can still fail from roughly 33/80 HP if the deck lacks enough burst/AoE to kill a segment before repeated 34+ incoming turns.
- Prefer entering with either a large HP buffer, strong AoE/frontload, or multiple defensive resources; one Plating potion is not a complete answer to the offset multi-attack cycle.
- Bloodletting+ into Whirlwind+ can kill all A10 segments together on turn 1 if X reaches 7; this avoids Reattach entirely and is worth 3 HP.
