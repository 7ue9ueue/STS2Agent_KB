# Skulking Colony

## Overview

**Skulking Colony** is an Elite Enemy found in the Underdocks.

## Stats

### Skulking Colony
- **HP**: 70 (Asc8+: 75)
- **Appears in**: Debut Underdocks

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Smash |  | Deals 12 damage. |
| _(Asc variant)_ | | Deals 13 (Asc9+) damage. |
| Zoom |  | Deals 14 damage and gains 10 Block. |
| _(Asc variant)_ | | Deals 16 (Asc9+) damage and gains 13 (Asc8+) Block. |
| Inertia |  | Deals 9 damage and gains 2 Strength. |
| _(Asc variant)_ | | Deals 11 (Asc9+) damage and gains 3 (Asc9+) Strength. |
| Piercing Stabs |  | Deals 7x2 damage. |
| _(Asc variant)_ | | Deals 8x2 (Asc9+) damage. |

## Pattern

The Skulking Colony repeats the same four moves in order:
1. Smash
1. Zoom
1. Inertia
1. Piercing Stabs
## Trivia

- Skulking Colony is male.

## Strategy Notes

<!-- Reserved for the meta-agent. Append lessons, combos, and heuristics here.

- Live A10 has Hardened Shell 15: the `HARDENED_SHELL_POWER` amount is remaining HP loss allowed this player turn. Stop attacking after it reaches 0.
- Akabeko/Sword Boomerang burst can waste damage into Hardened Shell; prefer just enough attacks to hit the 15 cap, then block or save cards for the next turn.
- If Zoom's Block prevents HP loss, a weak hand can be forced into potion use; enter with enough HP/potion equity to absorb Piercing Stabs after Inertia's Strength gain.
