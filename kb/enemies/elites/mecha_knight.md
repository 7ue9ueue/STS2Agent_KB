# Mecha Knight

## Overview

**Mecha Knight** is an Elite found in Glory.

## Stats

### Mecha Knight
- **HP**: 300 (Asc8+: 320)
- **Appears in**: Debut Glory

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Charge |  | Deals 25 damage. |
| _(Asc variant)_ | | Deals 30 (Asc9+) damage. |
| Flamethrower |  | Shuffles 4 Burns into your hand. |
| Windup |  | Gains 15 Block and 5 Strength. |
| Heavy Cleave |  | Deals 35 damage. |
| _(Asc variant)_ | | Deals 40 (Asc9+) damage. |

## Pattern

Starts with Charge and then repeats this cycle for the rest of combat:
- Flamethrower
- Windup
- Heavy Cleave

## Strategy Notes

<!-- Reserved for the meta-agent. Append lessons, combos, and heuristics here.
     Wiki facts live above the divider. -->
- Treat Heavy Cleave as a hard survival check after each Windup; Strength from repeated Windups makes later cycles lethal at low HP.
- Burns from Flamethrower can consume Buffer before Heavy Cleave. Exhaust Burns or cover their end-turn damage before relying on Lucky Tonic or other Buffer.
- Strip Artifact before the Heavy Cleave turn if possible so Weak/Vulnerable cards can both reduce the hit and shorten the fight.
