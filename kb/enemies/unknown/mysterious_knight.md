# Mysterious Knight

## Overview

**Mysterious Knight** is a normal Enemy encountered in the The Lantern Key event. It is a stronger version of the Flail Knight with innate Strength and Plating.

## Stats

### Mysterious Knight
- **HP**: 101 (Asc8+: 108)
- **Appears in**: Debut The Lantern Key

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Breaker |  | Gains 3 Strength. |
| Flail |  | Deals 9 damage x2. |
| _(Asc variant)_ | | Deals 10 (Asc9+) damage x2. |
| Ram |  | Deals 15 damage. |
| _(Asc variant)_ | | Deals 17 (Asc9+) damage. |

## Pattern

Opens with Ram, then randomly picks Breaker (equal chance), Flail (double chance), or Ram (double chance). Cannot use the same move twice in a row.
## Notes

- The Loot always contains the option to add the Lantern Key to the deck
