# Turret Operator

## Overview

**Turret Operator** is a weak Enemy encounter found in Glory, featuring a Living Shield and a Turret Operator.

## Stats

### Turret Operator
- **HP**: 41 (Asc8+: 51)
- **Appears in**: Debut Glory

### Living Shield
- **HP**: 55 (Asc8+: 65)
- **Appears in**: Debut Glory

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Unload! |  | Deals 3 damage x5. |
| _(Asc variant)_ | | Deals 4 (Asc9+) damage x5. |
| Loading |  | Gains 1 Strength. |

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Shield Slam |  | Deals 6 damage. |
| Smash |  | Deals 16 damage. Gains 3 Strength. |
| _(Asc variant)_ | | Deals 18 (Asc9+) damage. Gains 3 Strength. |

## Turret Operator

### Pattern
Follows a fixed 3-move cycle: Unload!, Unload!, Loading, then repeats.
## Living Shield

### Pattern
Uses Shield Slam every turn while the Turret Operator is alive. Once the Turret Operator is killed, switches to Smash every turn.

## Strategy Notes

<!-- Reserved for the meta-agent. Append lessons, combos, and heuristics here.
     Wiki facts live above the divider. -->
- Kill Living Shield FIRST to remove Rampart (stops 25 Block/turn on Turret Operator). Living Shield switches to Smash (16+3 Str) only when Turret Operator dies — so kill Living Shield before Turret Operator.
- Mr. Struggles + Mercury Hourglass passive chip fires BEFORE Rampart gives block — passives always deal HP damage to Turret Operator even through Rampart.
- Thorns potion vs Turret Operator Unload (5 hits): 3 Thorns × 5 = 15 reflected per Unload turn. Two Unload turns kills it from Thorns alone (37 HP − 15 − 15 = 7, then passive chip finishes).
- T3 = Loading (no hits, no Thorns proc). Pattern: Unload, Unload, Loading, repeat.
- Barricade or other retained-block engines can ignore Operator's repeated Unload after Shield dies; convert the retained block with Body Slam for a clean finish.
