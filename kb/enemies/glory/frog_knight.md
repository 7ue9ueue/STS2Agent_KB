# Frog Knight

## Overview

**Frog Knight** is a normal Enemy found in Glory.

## Stats

### Frog Knight
- **HP**: 191 (Asc8+: 199)
- **Appears in**: Debut Glory

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Tongue Lash |  | Deals 13 damage. Applies 2 Frail. |
| _(Asc variant)_ | | Deals 14 (Asc9+) damage. Applies 2 Frail. |
| Strike Down Evil |  | Deals 21 damage. |
| _(Asc variant)_ | | Deals 23 (Asc9+) damage. |
| For the Queen |  | Gains 5 Strength. |
| Beetle Charge |  | Deals 35 damage. |
| _(Asc variant)_ | | Deals 40 (Asc9+) damage. |

## Pattern

Opens with Tongue Lash, then follows a repeating cycle: Strike Down Evil, For the Queen, then a conditional check.

If the Frog Knight's HP is below half and it has not yet used Beetle Charge, it uses Beetle Charge once, then continues with Tongue Lash. Otherwise, it uses Tongue Lash and continues the cycle.
