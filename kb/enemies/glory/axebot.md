# Axebot

## Overview

**Axebot** is a normal Enemy found in Glory. It appears in pairs of 2.

## Stats

### Axebot
- **HP**: 40-44 (Asc8+: 42-46)
- **Appears in**: Debut Glory

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Boot Up |  | Gains 10 Block and 1 Strength. |
| The One-Two |  | Deals 5 damage 2 times. |
| _(Asc variant)_ | | Deals 6 (Asc9+) damage 2 times. |
| Sharpen |  | Gains 4 Strength. |
| Hammer Uppercut |  | Deals 8 damage. Applies 1 Weak and 1 Frail. |
| _(Asc variant)_ | | Deals 10 (Asc9+) damage. Applies 1 Weak and 1 Frail. |

## Pattern

Each turn, the Axebot randomly uses The One-Two (weight 2), Sharpen (cannot repeat), or Hammer Uppercut (weight 2).

When spawned mid-fight via Stock, it opens with Boot Up before entering the random pattern.
## In Party With

### Glory
- 2 Axebots
## Notes

- If a multi-hit Attack that doesn't target a single enemy (such as Sword Boomerang or Dagger Spray) kills an Axebot with Stock partway through the attack, the new Axebot will be summoned in the middle of the attack, and it can be hit by that same Attack.
    - This does _not_ occur for multi-hit Attacks that only hit a single enemy, like Twin Strike.
- When an Axebot with Stock dies during the enemy turn, such as by Poison or Thorns, the summoned Axebot will not do anything that turn, and it will use Boot Up on the following turn.
