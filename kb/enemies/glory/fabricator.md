# Fabricator

## Overview

**Fabricator** is a normal Enemy found in Glory. It summons bots to fight alongside it.

## Fabricator

### Pattern
Each turn, the Fabricator checks if it has fewer than 4 living allies. If so, it randomly uses either Fabricate or Fabricating Strike (equal weight). If it already has 4 or more allies, it uses Disintegrate instead.

The Fabricator avoids summoning the same bot consecutively.
