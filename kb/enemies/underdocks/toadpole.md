# Toadpole

## Overview

**Toadpole** is a normal Enemy found in the Underdocks.

## Stats

### Toadpole
- **HP**: 21-25 (Asc8+: 22-26)
- **Appears in**: Debut Underdocks

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Spike Spit |  | Deals 3 damage ×3. Removes 2 Thorns from self. |
| _(Asc variant)_ | | Deals 4 (Asc9+) damage ×3. Removes 2 Thorns from self. |
| Whirl |  | Deals 7 damage. |
| _(Asc variant)_ | | Deals 8 (Asc9+) damage. |
| Spiken |  | Gains 2 Thorns. |

## Pattern

Cycles through its moves in a fixed order: Whirl, then Spiken, then Spike Spit, repeating. In the Toadpoles (Weak) encounter, the front Toadpole starts with Spiken instead.
