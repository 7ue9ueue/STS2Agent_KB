# Corpse Slug

## Overview

**Corpse Slug** is a normal Enemy found in the Underdocks. It appears in groups of 2 or 3, and starts with Ravenous.

## Stats

### Corpse Slug
- **HP**: 25-27 (Asc8+: 27-29)
- **Appears in**: Debut Underdocks

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Whip Slap |  | Deals 3 damage ×2. |
| Glomp |  | Deals 8 damage. |
| _(Asc variant)_ | | Deals 9 (Asc9+) damage. |
| Goop |  | Applies 2 Frail. |

## Pattern

Cycles through its moves in a fixed order:
1. Whip Slap
1. Glomp
1. Goop
1. Back to Whip Slap

Each Corpse Slug in the encounter starts on a different move in the cycle, so they are staggered.
