# Haunted Ship

## Overview

**Haunted Ship** is a normal Enemy found in the Underdocks.

## Stats

### Haunted Ship
- **HP**: 63 (Asc8+: 67)
- **Appears in**: Debut Underdocks

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Ramming Speed |  | Deals 10 damage. Applies 1 Weak. |
| _(Asc variant)_ | | Deals 11 (Asc9+) damage. Applies 1 Weak. |
| Swipe |  | Deals 13 damage. |
| _(Asc variant)_ | | Deals 14 (Asc9+) damage. |
| Stomp |  | Deals 4 damage ×3. |
| _(Asc variant)_ | | Deals 5 (Asc9+) damage ×3. |
| Haunt |  | Shuffles 5 Dazed into your discard pile. |

## Pattern

Always uses Haunt first.

From the second turn onward, alternates between Ramming Speed and a random choice between Swipe or Swipe.
