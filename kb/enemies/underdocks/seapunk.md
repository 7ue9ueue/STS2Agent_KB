# Seapunk

## Overview

**Seapunk** is a normal Enemy found in the Underdocks.

## Stats

### Seapunk
- **HP**: 44-46 (Asc8+: 47-49)
- **Appears in**: Debut Underdocks

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Sea Kick |  | Deals 11 damage. |
| _(Asc variant)_ | | Deals 13 (Asc9+) damage. |
| Spinning Kick |  | Deals 2 damage ×4. |
| Bubble Burp |  | Gains 7 Block and 1 Strength. |
| _(Asc variant)_ | | Gains 8 (Asc8+) Block and 2 (Asc9+) Strength. |

## Pattern

Cycles through its moves in a fixed order:
1. Sea Kick
1. Spinning Kick
1. Bubble Burp
1. Back to Sea Kick
