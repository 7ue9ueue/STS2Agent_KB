# Sludge Spinner

## Overview

**Sludge Spinner** is a normal Enemy found in the Underdocks.

## Stats

### Sludge Spinner
- **HP**: 37-39 (Asc8+: 41-42)
- **Appears in**: Debut Underdocks

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Oil Spray |  | Deals 8 damage. Applies 1 Weak. |
| _(Asc variant)_ | | Deals 9 (Asc9+) damage. Applies 1 Weak. |
| Slam |  | Deals 11 damage. |
| _(Asc variant)_ | | Deals 12 (Asc9+) damage. |
| Rage |  | Deals 6 damage. Gains 3 Strength. |
| _(Asc variant)_ | | Deals 7 (Asc9+) damage. Gains 3 Strength. |

## Pattern

Always uses Oil Spray first. After that, randomly uses one of its three moves each turn. Cannot use the same move twice in a row.
