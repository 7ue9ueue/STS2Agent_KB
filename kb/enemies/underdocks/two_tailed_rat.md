# Two-Tailed Rat

## Overview

**Two-Tailed Rat** is a normal Enemy found in the Underdocks. Three appear together in the encounter.

## Stats

### Two-Tailed Rat
- **HP**: 17-21 (Asc8+: 18-22)
- **Appears in**: Debut Underdocks

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Scratch |  | Deals 8 damage. |
| _(Asc variant)_ | | Deals 9 (Asc9+) damage. |
| Disease Bite |  | Deals 6 damage. |
| _(Asc variant)_ | | Deals 7 (Asc9+) damage. |
| Screech |  | Applies 1 Frail. |
| Call for Backup |  | Summons a new Two-Tailed Rat. |

## Pattern

Each rat starts with a different move: one uses Scratch, one uses Disease Bite, and one uses Screech. After the first turn, each rat randomly picks from Scratch, Disease Bite, or Screech each turn. Cannot use the same move twice in a row.

After at least two turns, a rat may use Call for Backup to summon a new rat. Each rat can only summon once, and the group can summon up to three times total.
## Notes

- Despite its name, the Two-Tailed Rat doesn't visually have two tails and instead has one tail that is forked at the end.
- Fungi Beasts appear to be Two-Tailed Rats corpses with a fungal infection controlling them.
