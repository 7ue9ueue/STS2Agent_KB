# Cubex Construct

## Overview

**Cubex Construct** is a normal Enemy found in the Overgrowth and Glory.

## Stats

### Cubex Construct
- **HP**: 65 (Asc8+: 70)
- **Appears in**: Debut Overgrowth

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Charge Up |  | Gains 2 Strength. |
| Repeater Blast |  | Deals 7 damage. Gains 2 Strength. |
| _(Asc variant)_ | | Deals 8 (Asc9+) damage. Gains 2 Strength. |
| Expel Blast |  | Deals 5 damage ×2. |
| _(Asc variant)_ | | Deals 6 (Asc9+) damage ×2. |
| Submerge |  | Gains 15 Block. |

## Pattern

After it Charge Up it repeats this pattern for the remainder of fight:
- Repeater Blast
- Repeater Blast
- Expel Blast
## Notes

- Cubex Construct begins the fight with 13 Block and 1 Artifact, which negates the first debuff applied to it.
- Charge Up and each Repeater Blast each grant 2 Strength, so its damage ramps up significantly over time. After the opening Charge Up and two Repeater Blast uses, it will have gained 6 Strength.
- Expel Blast hits twice, making it benefit doubly from accumulated Strength.

- In Act 3 Glory fights, Cubex start with Str 2 and Artifact 1. Pattern: Repeater Blast (Attack+Empower) 3 times, then Expel Blast (5×2 = 11×2 at Str 6), then likely repeat. Str escalates +2 per Repeater Blast.
