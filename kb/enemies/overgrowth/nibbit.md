# Nibbit

## Overview

**Nibbit** is a normal Enemy found in the Overgrowth.

## Stats

### Nibbit
- **HP**: 42-46 (Asc8+: 44-48)
- **Appears in**: Debut Overgrowth

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Butt |  | Deals 12 damage. |
| _(Asc variant)_ | | Deals 13 (Asc9+) damage. |
| Hesitant Slice |  | Deals 6 damage. Gains 5 Block. |
| _(Asc variant)_ | | Deals 7 (Asc9+) damage. Gains 6 (Asc8+) Block. |
| Hiss |  | Gains 2 Strength. |
| _(Asc variant)_ | | Gains 3 (Asc9+) Strength. |

## Pattern

When alone (Nibbits Weak encounter), the Nibbit opens with Butt.

When paired (Nibbits Normal encounter), the front Nibbit opens with Hesitant Slice and the back Nibbit opens with Hiss.

After the opening move, all Nibbits cycle in a fixed order: Butt, then Hesitant Slice, then Hiss, and repeat.
## Notes

- Hiss grants 2 Strength, which increases the damage of all subsequent attacks. Over a long fight, Nibbits can accumulate significant Strength.
- In the paired encounter, the two Nibbits start on different points in their cycle, so they use different moves each turn.
