# Vine Shambler

## Overview

**Vine Shambler** is a normal Enemy found in the Overgrowth.

## Stats

### Vine Shambler
- **HP**: 61 (Asc8+: 64)
- **Appears in**: Debut Overgrowth

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Grasping Vines |  | Deals 8 damage. Applies 1 Tangled. |
| _(Asc variant)_ | | Deals 9 (Asc9+) damage. Applies 1 Tangled. |
| Swipe |  | Deals 6 damage ×2. |
| _(Asc variant)_ | | Deals 7 (Asc9+) damage ×2. |
| Chomp |  | Deals 16 damage. |
| _(Asc variant)_ | | Deals 18 (Asc9+) damage. |

## Pattern

Follows a fixed repeating cycle:
1. Swipe
1. Grasping Vines
1. Chomp
1. Back to Swipe
