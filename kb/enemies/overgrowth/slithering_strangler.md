# Slithering Strangler

## Overview

**Slithering Strangler** is a normal Enemy found in the Overgrowth.

## Stats

### Slithering Strangler
- **HP**: 53-55 (Asc8+: 54-56)
- **Appears in**: Debut Overgrowth

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Constrict |  | Applies 3 Constrict. |
| Thwack |  | Deals 7 damage. Gains 5 Block. |
| _(Asc variant)_ | | Deals 8 (Asc9+) damage. Gains 5 Block. |
| Lash |  | Deals 12 damage. |
| _(Asc variant)_ | | Deals 13 (Asc9+) damage. |

## Pattern

Always opens with Constrict. After that, it randomly picks Thwack or Lash (equal chance, can repeat). After either attack, it always uses Constrict again, then returns to the random pick.

The pattern is:
*Constrict
- 50/50 chance to use Lash or Thwack
- Constrict
- 50/50 chance to use Lash or Thwack
