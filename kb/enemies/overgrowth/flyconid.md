# Flyconid

## Overview

**Flyconid** is a normal Enemy found in the Overgrowth.

## Stats

### Flyconid
- **HP**: 47-49 (Asc8+: 51-53)
- **Appears in**: Debut Overgrowth

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Weakening Spores |  | Applies 2 Vulnerable. |
| Frail Spores |  | Deals 8 damage. Applies 2 Frail. |
| _(Asc variant)_ | | Deals 9 (Asc9+) damage. Applies 2 Frail. |
| Smash |  | Deals 11 damage. |
| _(Asc variant)_ | | Deals 12 (Asc9+) damage. |

## Pattern

On its first turn, it randomly picks Frail Spores (2/3 chance) or Smash (1/3 chance).

After that, each turn it randomly picks one of three moves: Weakening Spores (3/6 chance), Frail Spores (2/6 chance), or Smash (1/6 chance). It cannot use the same move twice in a row.
## Notes

- Weakening Spores applies Vulnerable, causing you to take 50% more damage from attacks. This is especially dangerous when paired with other enemies.
- Frail Spores applies Frail, reducing your Block gained by 25%.
- Flyconid cannot open with Weakening Spores on the first turn.
