# Mawler

## Overview

**Mawler** is a normal Enemy found in the Overgrowth.

## Stats

### Mawler
- **HP**: 72 (Asc8+: 76)
- **Appears in**: Debut Overgrowth

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Rip and Tear |  | Deals 14 damage. |
| _(Asc variant)_ | | Deals 16 (Asc9+) damage. |
| Roar |  | Applies 3 Vulnerable. |
| Claw |  | Deals 4 damage ×2. |
| _(Asc variant)_ | | Deals 5 (Asc9+) damage ×2. |

## Pattern

Always opens with Claw.

After that, each turn it randomly picks one of three moves with equal chance: Rip and Tear, Roar, or Claw. It cannot use the same move twice in a row, and Roar can only be used once per fight.
## Trivia

- The Mawler's name and appearance reference The Maw, an enemy from the original Slay the Spire.
