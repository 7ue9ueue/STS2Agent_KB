# Shrinker Beetle

## Overview

**Shrinker Beetle** is a normal Enemy found in the Overgrowth.

## Stats

### Shrinker Beetle
- **HP**: 38-40 (Asc8+: 40-42)
- **Appears in**: Debut Overgrowth

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Shrinker |  | Applies Shrink. |
| Chomp |  | Deals 7 damage. |
| _(Asc variant)_ | | Deals 8 (Asc9+) damage. |
| Stomp |  | Deals 13 damage. |
| _(Asc variant)_ | | Deals 14 (Asc9+) damage. |

## Pattern

Always opens with Shrinker, then alternates between Chomp and Stomp.
## Trivia

- This enemy resembles the Preserved Insect from the original Slay the Spire, which made Elites smaller when in use.
- The effect applied by its Shrinker ability is the same as that of the Beetle Juice.
