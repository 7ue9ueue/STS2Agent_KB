# Fuzzy Wurm Crawler

## Overview

**Fuzzy Wurm Crawler** is a normal Enemy found in the Overgrowth.

## Stats

### Fuzzy Wurm Crawler
- **HP**: 55-57 (Asc8+: 58-59)
- **Appears in**: Debut Overgrowth

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Acid Goop |  | Deals 4 damage. |
| _(Asc variant)_ | | Deals 6 (Asc9+) damage. |
| Inhale |  | Gains 7 Strength. |

## Pattern

Repeats a fixed three-turn cycle:
1. Acid Goop
1. Inhale
1. Acid Goop

Then the cycle starts over.
## Notes

- Inhale grants 7 Strength, which greatly increases the damage of the following Acid Goop. However, this Strength also carries over to future cycles, so each Acid Goop hits harder than the last.
- The base damage of Acid Goop is low (4), but after the first Inhale it deals 11 damage, and it continues to scale with each cycle.
