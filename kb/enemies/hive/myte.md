# Myte

## Overview

**Myte** is a normal Enemy found in Hive.

## Stats

### Myte
- **HP**: 61-67 (Asc8+: 64-69)
- **Appears in**: Debut Hive

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Toxic Cornucopia |  | Adds 2 Toxic to your hand. |
| Bite |  | Deals 13 damage. |
| _(Asc variant)_ | | Deals 15 (Asc9+) damage. |
| Suck |  | Deals 4 damage. Gains 2 Strength. |
| _(Asc variant)_ | | Deals 6 (Asc9+) damage. Gains 3 (Asc9+) Strength. |

## Pattern

Mytes always appear in pairs. One opens with Toxic Cornucopia and the other opens with Suck. After that, each one cycles through Toxic Cornucopia, Bite, Suck in order, repeating.
