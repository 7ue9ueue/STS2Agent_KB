# Hunter Killer

## Overview

**Hunter Killer** is a normal Enemy found in Hive.

## Stats

### Hunter Killer
- **HP**: 121 (Asc8+: 126)
- **Appears in**: Debut Hive

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Tenderizing Goop |  | Applies 1 Tender. |
| Bite |  | Deals 17 damage. |
| _(Asc variant)_ | | Deals 19 (Asc9+) damage. |
| Puncture |  | Deals 7 damage ×3. |
| _(Asc variant)_ | | Deals 8 (Asc9+) damage ×3. |

## Pattern

Always opens with Tenderizing Goop. After that, randomly uses Bite or Puncture, with Puncture being twice as likely. Bite cannot be used twice in a row.
