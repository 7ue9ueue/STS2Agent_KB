# Thieving Hopper

## Overview

**Thieving Hopper** is a normal Enemy found in Hive.

## Stats

### Thieving Hopper
- **HP**: 79 (Asc8+: 84)
- **Appears in**: Debut Hive

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Thievery |  | Deals 17 damage. Steals a card from your deck. |
| _(Asc variant)_ | | Deals 19 (Asc9+) damage. Steals a card from your deck. |
| Flutter |  | Gains Flutter (takes 50% less Attack damage; must be hit 5 times to Stun it). |
| Hat Trick |  | Deals 21 damage. |
| _(Asc variant)_ | | Deals 23 (Asc9+) damage. |
| Nab |  | Deals 14 damage. |
| _(Asc variant)_ | | Deals 16 (Asc9+) damage. |
| Escape |  | Flees the combat, taking any stolen cards with it. |

## Pattern

The Thieving Hopper follows a fixed sequence: Thievery (stealing a card and dealing damage), then Flutter (gaining damage reduction), then Hat Trick, then Nab, and finally Escape (fleeing the fight permanently).

If it escapes, any cards it stole are lost for the rest of the run. Killing it before it escapes returns the stolen cards.
## Notes

The card stolen by Swipe follows a set order of priority:

1. Non-Imbued Uncommon cards

2. Non-Imbued Rare, Common, and Event cards

3. Non-Imbued Basic and Quest cards

4. Ancient cards or cards that are Imbued

5. Status, Curse, and Token cards

Only cards currently in the player's draw or discard pile when the move is used will be selected. This means under the right circumstances the move can be utilized to remove Eternal cards from the player's deck.

## Strategy Notes

<!-- Reserved for the meta-agent. Append lessons, combos, and heuristics here.
     Wiki facts live above the divider. -->
- Steal priority: Uncommon first. Unplayed hand cards are discarded before Swipe resolves, so play key Uncommons or be ready to kill before Escape; simply holding them in hand does not protect them.
- Kill before Escape (T5) to recover the stolen card. If you kill it by T4, it always returns the card.
- T2 Flutter is a free offense turn (no damage) — use it to Bash+/Tremble for Vul setup. Multi-hit + Vul still effective through Flutter's 50% reduction.
- On T3 Hat Trick, five attack-damage hits into Flutter stun it and can prevent the 21-damage attack.
- Exterminate supplies four Flutter hits by itself; pair it with any other Attack hit to stun the T3 Hat Trick turn.
- If energy is already spent, 0-cost draw can still be worth playing before ending turn to shape the next hand for Flutter hits and the pre-Escape kill.
