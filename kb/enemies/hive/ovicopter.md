# Ovicopter

## Overview

**Ovicopter** is a normal Enemy found in the Hive. It summons Tough Eggs that hatch into Hatchlings.

## Stats

### Ovicopter
- **HP**: 124-130 (Asc8+: 126-132)
- **Appears in**: Debut Hive

### Tough Egg
- **HP**: 14-18 (Asc8+: 15-19)
- **Appears in**: Debut Hive

## Ovicopter

### Pattern
Always opens with Lay Eggs to summon 3 Tough Eggs, then uses Smash, then Tenderizer.

After Tenderizer, it checks how many allies it has. If it has 3 or fewer living allies, it uses Lay Eggs again. If it has more than 3, it uses Nutritional Paste instead. Then the cycle continues with Smash and Tenderizer.
## Tough Egg

The Tough Egg is a minion summoned by Ovicopter via Lay Eggs. It starts with the Hatch power, which counts down over turns.

### Pattern
The Tough Egg waits while its Hatch timer counts down. When the timer expires, it uses Hatch to transform into a Hatchling with new HP, then uses Nibble every turn after.
## Notes

- Hatch timer starts at 2 if summoned during the enemy turn, or 1 during the player turn.
