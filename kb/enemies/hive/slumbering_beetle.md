# Slumbering Beetle

## Overview

**Slumbering Beetle** is a normal Enemy found in Hive.

## Stats

### Slumbering Beetle
- **HP**: 86 (Asc8+: 89)
- **Appears in**: Debut Hive

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Snore |  | Does nothing. Asleep. |
| Roll Out |  | Deals 16 damage. Gains 2 Strength. |
| _(Asc variant)_ | | Deals 18 (Asc9+) damage. Gains 2 Strength. |

## Pattern

The Slumbering Beetle starts asleep and uses Snore each turn while it has Slumber. Once Slumber wears off (after taking turns or losing HP 3 times), it wakes up, loses its Plating, and uses Roll Out every turn.
## In Party With

### Hive
- + Bowlbug (Rock) + Bowlbug (Silk)
