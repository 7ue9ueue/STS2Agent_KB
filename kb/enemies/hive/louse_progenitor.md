# Louse Progenitor

## Overview

**Louse Progenitor** is a normal Enemy found in Hive.

## Stats

### Louse Progenitor
- **HP**: 134-136 (Asc8+: 138-141)
- **Appears in**: Debut Hive

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Web Cannon |  | Deals 9 damage. Applies 2 Frail. |
| _(Asc variant)_ | | Deals 10 (Asc9+) damage. Applies 2 Frail. |
| Curl and Grow |  | Gains 14 Block. Gains 5 Strength. |
| _(Asc variant)_ | | Gains 18 (Asc8+) Block. Gains 5 Strength. |
| Pounce |  | Deals 14 damage. |
| _(Asc variant)_ | | Deals 16 (Asc9+) damage. |

## Pattern

The Louse Progenitor repeats a fixed three-move cycle: Web Cannon, then Curl and Grow, then Pounce, and back to the start.

## Strategy Notes

<!-- Reserved for the meta-agent. Append lessons, combos, and heuristics here.
     Wiki facts live above the divider. -->
- T2 Curl and Grow (no attack) is a free offense turn — go all-in damage.
- After Curl and Grow it has +5 Str; Pounce becomes 14+5=19 dmg. Kill before T4 or need 20+ Block.
- Web Cannon applies 2 Frail (−25% Block effectiveness) — if Str is high, this compounds. Kill by T3 to avoid second Frail cycle.
- If lethal is not guaranteed, respect the 19-damage Pounce; spending the no-attack turn only on setup can still lead to a large HP loss.
