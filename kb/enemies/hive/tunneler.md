# Tunneler

## Overview

**Tunneler** is a normal Enemy found in Hive.

## Stats

### Tunneler
- **HP**: 87 (Asc8+: 92)
- **Appears in**: Debut Hive

### Attacks

| Name | Intent | Effect |
| --- | --- | --- |
| Bite |  | Deals 13 damage. |
| _(Asc variant)_ | | Deals 15 (Asc9+) damage. |
| Burrow |  | Gains Burrowed and 32 Block. |
| _(Asc variant)_ | | Gains Burrowed and 37 (Asc8+) Block. |
| Attack from Below |  | Deals 23 damage. |
| _(Asc variant)_ | | Deals 26 (Asc9+) damage. |
| Emerging Strike |  | Stunned. Does nothing this turn. |

## Pattern

Opens with Bite, then uses Burrow. While burrowed, uses Attack from Below every turn.

If its Block is fully broken while burrowed, it uses Emerging Strike, then starts the cycle over with Bite.
## In Party With

### Hive
- Solo (Weak)
- + 1 Bowlbug (Egg) or Bowlbug (Silk) (Normal)

- Save burst damage for the Burrow turn; on A10, fully breaking the 37 Block prevents the 26-damage Attack from Below with a stun.
