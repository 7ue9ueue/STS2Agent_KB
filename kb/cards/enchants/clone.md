# Clone

## Effect

This card can be duplicated at Rest Sites.

## Sources And Targets

| Source | Source type | Target |
|---|---|---|
| Pael's Growth | Ancient Relic: Pael | 1 card |

## Notes

- Cloned copies keep Clone, so repeat rest-site cloning scales the number of copies rapidly.
- Cards can only have one enchantment.
