# Tezcatara's Ember

## Effect

Costs 0 and gains Eternal.

## Sources And Targets

| Source | Source type | Target |
|---|---|---|
| Nutritious Soup | Ancient Relic: Tezcatara | All Strikes in your Deck |

## Notes

- Ultimate Strikes are not enchanted by Nutritious Soup.
- Already enchanted Strikes do not receive Tezcatara's Ember because each card can only have one enchantment.
