# Imbued

## Effect

This card is played automatically at the start of each combat.

## Sources And Targets

| Source | Source type | Target |
|---|---|---|
| Electric Shrymp | Ancient Relic: Orobas | 1 Skill |

## Notes

- The Imbued Skill is played after other start-of-combat effects and after the opening hand is drawn.
- Electric Shrymp places the Imbued Skill at the bottom of the Draw Pile, so it usually does not consume an opening draw.
- X-cost Skills use the current X value but do not spend Energy when Imbued plays them.
- Cards can only have one enchantment.
